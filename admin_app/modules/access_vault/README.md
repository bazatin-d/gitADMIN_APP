# Модуль «Доступы»

Корпоративное хранилище доступов: сайты, сервисы, соцсети, почта и архив.

## Состав

- `module.php` — регистрация вкладки `access_vault`.
- `pages/index.php` — интерфейс модуля.
- `actions.php` — POST-действия.
- `service.php` — валидация, генерация, SMTP, CSV.
- `repository.php` — SQL-слой.
- `crypto.php` — шифрование/расшифровка паролей.

## Требования

1. Выполнить миграцию `database/migrations/2026_05_27_010_create_access_vault.sql`.
2. В `config.php` добавить константу `ACCESS_VAULT_KEY` длиной минимум 32 символа.
3. В карточке пользователя включить нужные права для менеджеров.

Пароли в БД хранятся только в зашифрованном виде.
