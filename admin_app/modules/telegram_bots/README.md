# Модуль «Telegram-боты»

Первый слой модуля:

- подключение нескольких Telegram-ботов;
- шифрование токенов через ACCESS_VAULT_KEY;
- установка webhook на публичный файл `/telegram_bot_webhook.php`;
- сохранение подписчиков;
- сохранение входящих/исходящих сообщений;
- ответ на `/start`;
- журнал событий.

Следующие шаги: рассылки, теги, сегменты, сценарии.
