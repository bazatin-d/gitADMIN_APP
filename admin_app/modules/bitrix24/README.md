# Модуль Bitrix24

Модуль содержит слой интеграции с Bitrix24.

## Файлы

- `config.php` — загрузка настроек из `oca_settings`, объявление констант Bitrix24.
- `api.php` — безопасная обёртка REST-запросов, маскирование секретов, ротация логов.
- `service.php` — сценарии синхронизации результата теста: контакт, промежуточная сделка, финальная сделка.
- `actions.php` — пустой модульный обработчик для единого dispatcher.
- `repository.php` — резерв под SQL-слой.

## Совместимость

Старые файлы в `b24_integration/` оставлены прокладками:

- `b24_config.php`
- `b24_api.php`
- `sync_test.php`

Поэтому существующие вызовы из `save.php`, `save_progress.php`, результатов и ручного ввода продолжают работать.
