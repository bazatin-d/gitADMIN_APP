# Модуль results

Модуль отвечает за список результатов АСР и карточку результата.

## Что уже вынесено

- страница списка результатов;
- страница детализации/сравнения;
- отправка письма клиенту;
- восстановление записи из спама;
- ручная повторная синхронизация с Bitrix24;
- редактирование результата;
- удаление результата.

## Что пока оставлено в legacy

- CSV-экспорт и CSV-импорт — будущий модуль import_export;
- ручное создание результата — будущий модуль manual_input;
- часть helper-функций внутри страниц, чтобы не менять поведение графиков и таблицы за один шаг.
