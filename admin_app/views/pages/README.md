# Legacy pages

Эта папка оставлена только для обратной совместимости со старыми подключениями.

Реальные страницы находятся в:

```text
admin_app/modules/{module_id}/pages/
```

Правило разработки:

```text
Новую верстку, формы и логику страниц добавлять только в модульные pages.
Файлы из этой папки не расширять.
```

Если нужна новая вкладка админки, создаём модульный `module.php` и указываем там путь к странице.
