# SQL-миграции Admin App

Здесь лежат структурные изменения БД. Файлы выполняются вручную через phpMyAdmin или консоль MySQL.

## Порядок

1. Сначала выполнить `2026_05_27_000_create_app_migrations.sql`.
2. Затем выполнять нужные миграции по порядку.
3. После успешного выполнения каждая миграция сама добавляет запись в `app_migrations`.

## Важно

- Миграции не запускаются автоматически из PHP.
- Перед выполнением делать экспорт БД.
- Не менять уже выполненные файлы миграций. Для новых изменений создавать новый файл.
