# AI Workflow

Рабочий процесс для задач с ИИ.

## Маленькая задача

Пример: «в модуле Доступы не прокручивается окно ресурса».

Прислать:

```text
docs/project_graph.md
docs/file_map.md
admin_app/modules/access_vault/
assets/admin/modules/access_vault/
```

## Задача с меню

Прислать:

```text
docs/project_graph.md
admin_app/modules/menu/
admin_app/views/layout.php
assets/admin/js/admin.js
```

## Задача с пользователями и правами

Прислать:

```text
docs/project_graph.md
admin_app/modules/users/
admin_app/core/permissions.php
admin_app/views/partials/modals.php
assets/admin/js/admin.js
```

## Задача непонятно где

Прислать:

```text
docs/project_graph.md
docs/file_map.md
docs/generated_file_tree.md
скриншот
описание действия по шагам
```

## Когда нужен весь проект

Весь проект нужен редко:

- большой рефакторинг архитектуры;
- поиск сквозной ошибки;
- проверка безопасности всего приложения;
- перенос домена/БД;
- подготовка релиза.
